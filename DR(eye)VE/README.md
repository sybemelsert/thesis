# DR(eye)VE
This folder includes relevant driving-gaze data extracted from the official [DR(eye)VE](https://aimagelab-legacy.ing.unimore.it/imagelab/page.asp?IdPage=8) dataset, developed by researchers at the University of Modena and Reggio Emilia.

## Folders

### *modified*
Serves as the target directory for the [*video_converter.py*](video_converter.py)-file for modified video segments.

### *original*
Stores the untouched *.avi* video files of the DR(eye)VE dataset.

### *txt*
Includes the original *.txt*-files to extract correct gaze coordinates for each of the matching video frames.
